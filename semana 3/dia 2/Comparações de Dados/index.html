Comparação de Dados

  • (Recebe) = - 1 Igual, tem o valor de RECEBER
  • (Igual) == - 2 Iguais, compara valores, mas por ter tipagem fraca, sendo string ou numero, serão iguais.
  • (Identico) === - 3 Iguais, compara valores, verifica se tanto o valor quanto o tipo, é IDENTICO
  • (Diferente) != - 1 exclamação + 1 Igual, verifica se são diferentes
  • (Diferente) !== - 1 exclamação + 2 Iguais, verifica se são diferentes, incluindo valor
  • (maior / menor) > / < - verifica se é maior ou menor
  • (maior / menor) > / < - 1 exclamação + 2 Iguais, verifica se são diferentes, incluindo valor
