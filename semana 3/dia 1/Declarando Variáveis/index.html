Declarando Variáveis

  • LET

    • LET, uma variavel normal, posso substituir a informação quantas vezes quiser.

  • CONST

    • Const só recebe valor 1 única vez.

  • VAR

    • VAR funciona exatamente como o LET

  • CAMEL CASE

    • Variavel com 2 nomes, como é chamada, Camel Case.
      primeira palavra minuscula, (my) e a segunda com a primeira letra, maiuscula (Age), myAge
